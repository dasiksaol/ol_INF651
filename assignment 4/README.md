A simple web app that lets you add and remove movies from a watchlist.
it utilizes array and function of javascript. 

Manual:
1. Type a movie name and click Add Movie (or press Enter).
2. It shows up in your list below.
3. If you wish to remove a movie, You can delete a movie by clicking the trash icon.

Functionality:
All movies are stored in an array called movieArray.
addMovie() adds new movies to the array.
displayMovies() updates what you see on the page.
removeMovie() deletes a movie from the array.
