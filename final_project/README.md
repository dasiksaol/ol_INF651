================================================================

Typingtest is a browser-based typing test built using HTML, CSS and JavaScript. It demonstrates core JS concepts such as DOM, event handling, timers, conditional logics and arrays. 

The application allows users to practice typing, track their performance, and view their best scores. 

To start testing, select your preferred mode and time, then simply click "Start" and start typing. 
================================================================

Features:

+ Start and Reset buttons
+ Words mode (random unique words)
+ Sentence mode (paragraph)
+ Timer starts only when user first type
+ Live stats:
  - WPM 
  - Raw WPM
  - Accuracy
  - Char Count
  - Errors

Stats Page:
+ Show: 
  - Best WPM
  - Best Accuracy
  - Total Tests
+ Loads saved data from LocalStorage

Theme:
+ Light
+ Dark
+ Dracula

About Page:
+ Description of web app
